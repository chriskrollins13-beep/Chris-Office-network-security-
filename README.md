# Office Network Security Project

## Overview
Designed and configured a secure office network using Cisco Packet Tracer.

## Features
- Department-based network segmentation (HR, Sales, IT)
- Access Control Lists (ACLs) to restrict inter-department traffic
- DNS and server configuration
- Switch port security implementation

## Technologies Used
- Cisco Packet Tracer
- Networking (IP addressing, routing, ACLs)

## Testing
- Verified connectivity within departments
- Blocked communication between HR and Sales using ACLs
